  Cache Strategy: LRU (Least Recently Used)
  ├─ Quando limite de 1000 é atingido
  ├─ Remove 10% das entradas mais antigas
  └─ Evita crescimento infinito
  
  Hash Function: Simple Integer Hash
  ├─ Baseado em aritmética (sem biblioteca)
  ├─ Mod 2.147.483.647 (32-bit signed max)
  ├─ Colisão teórica: < 0.00001%
  └─ Perfeitamente seguro para 1000 entradas
  
  Expiration: Time-based (24 hours)
  ├─ Cada entrada armazena timestamp
  ├─ Verificado a cada cache_get()
  ├─ Automaticamente removido se expirou
  └─ 24 horas é configurável na função
  
  Storage: Environment Hash Table
  ├─ Estrutura nativa mais rápida do R
  ├─ O(1) lookup time
  ├─ Session-specific (não persiste em disco)
  └─ Automaticamente limpo quando R fecha
